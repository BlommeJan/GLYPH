## Overview

GLYPH is a zero-dependency web application designed for raw text manipulation. It offers a distraction-free "void" for your thoughts and immediate export options. It requires no installation, no server, and no internet connection to function.

## Features

* **Single-File Architecture:** The entire app is contained within one HTML file.
* **Local Persistence:** Auto-saves content to `localStorage`. Close the tab, come back later, your text remains.
* **Drag & Drop:** Drag `.txt` or `.md` files directly into the editor to load them.
* **Client-Side Export:**
    * **PDF:** Auto-paginated, monospaced styling.
    * **MD:** Markdown format.
    * **TXT:** Plain text.
* **Privacy:** 100% offline. No data leaves your machine.

## Usage

1.  Clone the repo or download `glyph.html`.
2.  Open `glyph.html` in any modern web browser.
3.  Type.
